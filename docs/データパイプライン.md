# データパイプライン

都道府県データを追加するときや、PMTilesを再生成するときの手順。

## 都道府県データ追加手順

### 1. e-Stat から町丁目 Shapefile をダウンロード

```bash
# XX = 都道府県コード（01〜47、ゼロ埋め2桁）
curl -o /tmp/estat_XX.zip -L \
  "https://www.e-stat.go.jp/gis/statmap-search/data?dlserveyId=A002005212020&code=XX&coordSys=1&format=shape&downloadType=5"
```

都道府県コード早見表（主要なもの）:

| コード | 都道府県 |
|---|---|
| 01 | 北海道 |
| 13 | 東京都 |
| 14 | 神奈川県 |
| 23 | 愛知県 |
| 27 | 大阪府 |
| 40 | 福岡県 |

### 2. Shapefile を GeoJSON に変換

```bash
# /tmp/estat_XX/ に解凍
unzip /tmp/estat_XX.zip -d /tmp/estat_XX/

# mapshaper で GeoJSON に変換（8%簡略化、必要フィールドのみ）
mapshaper /tmp/estat_XX/r2kaXX.shp \
  -simplify 8% \
  -filter-fields KEY_CODE,PREF_NAME,CITY_NAME,S_NAME \
  -o format=geojson "D:/github/chizunurie/public/data/XX_chocho.geojson"
```

### 3. build-pmtiles.mjs に追記

`scripts/build-pmtiles.mjs` の `LAYERS` 配列に追加：

```js
const LAYERS = [
  // ... 既存エントリ ...
  { name: 'chocho', file: 'XX_chocho.geojson', minZoom: 8, maxZoom: 13 },
];
```

### 4. PMTiles を再生成

```bash
node scripts/build-pmtiles.mjs
```

## PMTiles 生成スクリプトの仕組み

`scripts/build-pmtiles.mjs` の処理フロー：

```
GeoJSON 読み込み
  ↓
geojson-vt でタイルインデックス作成（各レイヤー独立）
  ↓
日本のbbox（122.9–154.0°E、20.4–45.6°N）内のタイルを全zoom列挙
  ↓
各タイル座標で全レイヤーの getTile() を呼んでデータ取得
  ↓
vt-pbf でPBFエンコード → gzip圧縮 → MBTiles（SQLite）に書き込み
  ↓
pmtiles.exe convert で MBTiles → PMTiles
```

### MBTiles の Y 座標に注意

geojson-vt は XYZ（Slippy Map）形式、MBTiles は TMS形式（Y軸反転）。

```js
// MBTiles用にY座標を変換
function toTMS(z, x, y) { return (1 << z) - 1 - y; }
```

## 市区町村データ（municipalities_poly.geojson）の再生成

```bash
# 国土数値情報 N03-2024 をダウンロード（約600MB）
curl -o /tmp/n03.zip -L \
  "https://nlftp.mlit.go.jp/ksj/gml/data/N03/N03-2024/N03-20240101_GML.zip"

unzip /tmp/n03.zip N03-20240101.shp N03-20240101.dbf N03-20240101.shx N03-20240101.prj -d /tmp/n03/

# 市区町村コードでdissolve → 簡略化 → GeoJSON
mapshaper /tmp/n03/N03-20240101.shp \
  -dissolve N03_007 copy-fields=N03_001,N03_004,N03_005 \
  -simplify 2% \
  -filter-fields N03_001,N03_004,N03_005,N03_007 \
  -o format=geojson "D:/github/chizunurie/public/data/municipalities_poly.geojson"

# ラベル用セントロイド
mapshaper /tmp/n03/N03-20240101.shp \
  -dissolve N03_007 copy-fields=N03_001,N03_004,N03_005 \
  -points inner \
  -filter-fields N03_001,N03_004,N03_005,N03_007 \
  -o format=geojson "D:/github/chizunurie/public/data/municipalities.geojson"
```

N03 フィールド説明：
- `N03_001`: 都道府県名
- `N03_004`: 市区町村名（政令市の場合は市名）
- `N03_005`: 区名（政令指定都市の区のみ）
- `N03_007`: 行政区域コード（5桁、市区町村を一意に識別）

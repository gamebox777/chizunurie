### 2026-05-20

- **GPS現在地表示**
  - `GeolocateControl` を追加（現在地への地図移動・追従）

- **市区町村単位での塗り実装**
  - 国土数値情報 N03-2024 Shapefile（600MB）をダウンロード
  - mapshaper で市区町村コード（N03_007）でdissolve → 2%簡略化 → 12.3MB のGeoJSON生成
  - 全国1905市区町村がクリックで塗れるように

- **市区町村名ラベル表示**
  - 全国1905市区町村のセントロイドGeoJSON生成（358KB）
  - MapLibreの `localIdeographFontFamily` で日本語フォントを使用（グリフサーバー不要）
  - zoom 6以上で市区町村名を表示、政令指定都市は区名で表示

- **町丁目ラインの表示（東京・北海道）**
  - e-Stat 国勢調査2020 小地域データをダウンロード（東京: 5.4MB、北海道: 19.9MB）
  - mapshaper で8%簡略化・不要フィールド削除
  - zoom 8以上で町丁目境界線をフェードイン表示

- **クリック優先順位の設計**
  - zoom 8以上では町丁目 > 市区町村の優先順位でクリック判定
  - `queryRenderedFeatures` で最前面レイヤーのフィーチャを取得

- **ズームレベル表示**
  - `zoom` イベントを DOM に直接書き込む方式（React state を使わない）
  - state を使うとバッチ処理で更新が遅れるため

- **プロジェクト立ち上げ**
  - Next.js 16.2.6（App Router、TypeScript、Tailwind CSS v4）
  - MapLibre GL JS v5.24.0 を導入
  - `page.tsx` に `'use client'` + `dynamic(..., { ssr: false })` でクライアント限定ロード
  - React Strict Modeの二重実行対策：cleanup で `mapRef.current = null` を設定
  - 都道府県GeoJSON（japan.geojson、47都道府県）で白地図を表示
  - クリックで都道府県を塗る最初の実装
